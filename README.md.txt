# 🤖 AI Gold Trader XAUUSD

Торговый робот на основе PPO (Proximal Policy Optimization) для торговли золотом.

## Features
- 🔥 Алгоритм PPO из stable-baselines3
- 📊 Интеграция с MetaTrader 5
- 🎨 Современный UI на Streamlit
- 🤖 Авто-расчёт лота
- 💾 Авто-сохранение лучших моделей
